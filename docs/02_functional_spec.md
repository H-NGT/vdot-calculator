# VDOT Calculator - 機能仕様書

## 1. 画面構成

### 1.1 メイン画面

```
┌─────────────────────────────────┐
│           🏃‍♂️ ロゴ              │
│      マラソンタイム予測          │
│     ダニエルズ式(VDOT)で予測     │
├─────────────────────────────────┤
│  ペース（キロあたり）            │
│  ┌──────┐  ┌──────┐            │
│  │ 5 分 │  │30 秒 │            │
│  └──────┘  └──────┘            │
├─────────────────────────────────┤
│  走行距離                       │
│  [5K] [10K] [ハーフ]            │
│  ┌─────────────────┐            │
│  │     10      km  │            │
│  └─────────────────┘            │
├─────────────────────────────────┤
│       [ 予測する → ]            │
├─────────────────────────────────┤

│  ┌─────────────────────┐        │

```

---

## 2. 入力仕様

### 2.1 ペース入力

| 項目 | 仕様 |
|-----|------|
| 分 | ドロップダウン選択、1〜10 |
| 秒 | ドロップダウン選択、0〜59 |
| 初期状態 | 未選択（プレースホルダー表示）|

### 2.2 距離入力

| 項目 | 仕様 |
|-----|------|
| タイプ | 数値入力 |
| 範囲 | 1〜50 km |
| 精度 | 小数第4位まで（step=0.0001）|
| プリセット | 5K(5), 10K(10), ハーフ(21.0975) |

---

## 3. 出力仕様

### 3.1 VDOT 値

| 項目 | 仕様 |
|-----|------|
| 形式 | 数値（小数第1位）|
| 例 | 40.0, 55.3 |

### 3.2 レースタイム予測

| 距離 | 表示形式 |
|-----|---------|
| 5K | MM:SS |
| 10K | MM:SS または H:MM:SS |
| ハーフマラソン | H:MM:SS |
| フルマラソン | H:MM:SS + ペース |

---

## 4. バリデーション

### 4.1 エラー条件

| 条件 | メッセージ |
|-----|----------|
| ペース未入力 | 「ペースを入力してください」|
| 距離未入力 | 「走行距離を入力してください」|
| 距離が範囲外 | HTML5 標準バリデーション |

### 4.2 エラー表示

- 結果エリアに赤色で表示
- 背景色: rgba(255, 107, 107, 0.1)
- ボーダー: 赤色

---

## 5. インタラクション

### 5.1 プリセットボタン

| 操作 | 挙動 |
|-----|------|
| クリック | 距離フィールドに値を設定、ボタンをアクティブ化 |
| 他ボタンクリック | 前のアクティブを解除 |

### 5.2 予測ボタン

| 操作 | 挙動 |
|-----|------|
| クリック | VDOT 計算実行、結果を表示 |
| ホバー | Y 方向に -2px 移動、シャドウ拡大 |

### 5.3 結果表示

| 操作 | 挙動 |
|-----|------|
| 計算成功 | フェードインアニメーション（0.4s）|
| カードホバー | Y 方向に -2px 移動 |
