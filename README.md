# Hello World Web App

シンプルな「Hello World」を表示する Web アプリケーションです。

## プレビュー

モダンなグラデーション背景とグラスモーフィズム効果を使用したデザインで「Hello World」を表示します。

## ファイル構成

```
├── index.html    # メイン HTML ファイル
├── style.css     # スタイルシート
├── .gitignore    # Git 除外ファイル
└── README.md     # このファイル
```

## 使い方

`index.html` をブラウザで開くだけで動作します。

```bash
open index.html
```
